# Si Paling Antek: Japanese
Instruksikan dirimu belajar bahasa baru.

Website for learning Japanese for Indonesia gamified. The project is currently under development.
Feaure you could expect:
- Ranking system, time and/ score based.
- Typing kanji in romaji and listening to the prnonounciations.
- Specialize training to help users better remember the kana they often forgot.

This is a passion project to improve my programming skills and also learn Japanese.
